# zeroG-KIT

**Your pocket toolbox.** A CLI toolkit built to remove everyday friction from your computer workflow.

## What it does

zeroG-KIT puts the tools you actually use into a single terminal interface — no browser tabs, no searching, no installs per task.

| # | Tool | Description |
|---|------|-------------|
| 1 | Clean Temp & Prefetch | Wipes %TEMP%, Windows\Temp and Prefetch in one shot |
| 2 | Clipboard History | Shows current clipboard contents |
| 3 | Image Converter | Convert any image to jpg, png, bmp, etc |
| 4 | Password Generator | Cryptographically secure random passwords |
| 5 | Hash / Checksum | SHA-256 hash of any string |
| 6 | Unit Converter | Temperature, weight, length, data size |
| 7 | QR Code Generator | Generate QR codes from URLs or text |
| 8 | Date & Time | Current date and time |
| 9 | System Info | OS, RAM, CPU, Disk usage |
| 10 | Password Strength | Check how strong a password is |
| 11 | Python Vulnerability Scanner | Scan .py files for unsafe patterns |
| 12 | Encryption Tool | AES encrypt/decrypt text with a key |

## Install

​```bash
git clone https://github.com/otzpt/zeroG-KIT
cd zeroG-KIT
pip install -r requirements.txt
python main.py
​```

## Built with

Python — `Pillow`, `psutil`, `qrcode`, `cryptography`, `hashlib`, `secrets`
